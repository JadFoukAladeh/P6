Environmental Setup

Software Download & Installation

For the purpose of testing this project the users are expected to be running Windows 10. This is to ensure the same testing environment on all computers to minimize errors in testing and findings. This section of the test plan will guide the users through the environment setup procedure that is required. Following this procedure will allow the user to be able to:

  • Launch a server from the Raspberry Pi
  • Connect remotely to a Raspberry Pi through local network and through VPN using VNC and SSH
  • Communicate with the server from a virtual machine
  • Use github to obtain the current code for the repository to be used for testing
There are 5 main software packages that need to be downloaded, which are as follows:
  1. Ubuntu 24.04 LTS - Image of Ubuntu being used
  2. Oracle VM VirtualBox
  3. VNC
  4. PuTTY
  5. Ivanti Secure Access Client

Ubuntu 24.04 LTS

  1. Open the following link: Ubuntu
  2. Download the latest Ubuntu package for Windows hosts - current version (24.04 LTS)

Oracle VM VirtualBox

  1. Open the following link: Oracle VM VirtualBox
  2. Download the latest VirtualBox platform packages for Windows hosts - current version (7.0.18)

PuTTY

Initial Virtual Machine Setup

  1. Install the Oracle VirtualBox from the executable file downloaded above
  2. Launch Oracle VirtualBox
  3. Press the "New" button to create a new virtual machine(VM)
  4. Name the VM something meaningful such as Project VI testing in the Name field
  5. In the ISO Image field press the drop down menu. Click Other and scroll to the Downloads folder and choose the ubuntu iso we downloaded in the previous step.
  6. Check the box to Skip Unattended Installation and click Next.
  7. Set a Base Memory of 8048MB and click next.
  8. Set a Disk Size of 100 GB and click next. Then click Finish.
  9. Launch the VM and go through initial setup. The remaining commands will all be executed in the VM.e
  10. The following commands will download and install LAMPP server which will be used to host an HTML server. This installation method will run the HTML server automatically upon the launch of the OS. Open the terminal and type the following commands:
    • sudo apt-get install net-tools
    • sudo apt-get install lamp-server^
    • sudo apt-get install git

Initial Server Setup

  1. sudo apt-get install code - this will install Visual Studio Code
  2. Open Visual Studio Code install the following extensions: click Ctrl+Shift+x to open the Extensions tab. Extensions can be installed by clicking the install button after searching for the extension needed.
    • GitLens - Git supercharged
    • Auto Close Tag
    • Babel JavaScript
    • Color Highlight
    • file-icons
    • GitHub Pull Requests
    • PHP Intelephense
    • Prettier - Code formatter
    • Remote - SSH
    • Remote - SSH: Editing Configuration Files
    • Remote Explorer
    • vscode-icons
  3. Within the terminal input the following cd /var/www/html
  4. touch index.html | sudo chmod -R 777 index.html - this will create a file which will be the index to the server linking all pages of the website. This file will be initialized with read, write, and execute priviliages.
  5. In the explorer tab in Visual Studio Code on the left hand side of the screen, you should now see index.html
  6. Open the index.html by clicking on it in Visual Studio Code. Type ! followed by Enter. This will create a dummy server that can be seen through a broweser
  7. In the terminal type hostname -I - this will give you the IPV4 address of the computer. Type this in the search bar of google chrome or firefox. This should launch a blank html page. Which means your server is now setup.
